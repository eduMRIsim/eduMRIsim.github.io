Features

  • Clinically Realistic UI & Workflow
  • 15+ Editable Scan Parameters
    • Contrast: TE, TR, flip angle, etc.
    • Geometry: slice thickness, scan plane, resolution, etc.
  • Generation of MR Images with Realistic:
    • Image Contrast
    • Signal-to-Noise Ratio
    • Scan Time
  • Fast Simulations with Analytical Signal Equations
  • Side-By-Side Image Comparison
  • Intuitive and Easy to Use GUI
  • Free of Cost

Upcoming Features
